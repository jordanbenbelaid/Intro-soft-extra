Short interview style technical questions:

EXERCISE 1 – String Reversal

Write a method that reverses a string without using built-in reverse methods.

Example:
Input: "hello"
Output: "olleh"

EXERCISE 2 – Count Character Frequency

Given a string, count how many times each character appears.

Example:
Input: "banana"
Output:
b = 1
a = 3
n = 2

EXERCISE 3 – Remove Duplicates from a List

Given a List<Integer>, return a new list with duplicates removed.

Example:
Input: [1, 2, 2, 3, 1]
Output: [1, 2, 3]

EXERCISE 4 – Custom Object Equality

Create a class Person with fields:
String name
int age
Ensure two Person objects are considered equal if both name and age match.
Demonstrate storing them in a HashSet.

EXERCISE 5 – Generic Box

Create a generic class Box<T> that:
Stores one value
Has methods set(T value) and get()
Demonstrate using Box with Integer and String.

EXERCISE 6 – Static Counter

Create a class Counter where each object increments a static counter.
Print how many objects were created.

EXERCISE 7 – Enum with Behavior

Create an enum Day with values:
MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY, SUNDAY
Add a method isWeekend() that returns true for Saturday and Sunday.

EXERCISE 8 – Exception Handling

Write a method divide(int a, int b) that:
Throws IllegalArgumentException if b is 0
Otherwise returns the result
Handle the exception in main.

EXERCISE 9 – Polymorphism

Create a base class Animal with method speak().
Create subclasses Dog and Cat that override speak().
Demonstrate polymorphism using a list of Animals.

EXERCISE 10 – Immutable Class

Design an immutable class ImmutableUser with:
private final fields
no setters
constructor validation
